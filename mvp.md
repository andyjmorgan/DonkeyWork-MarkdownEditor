# MVP

We're creating a beautiful markdown wysiwyg editor that supports the following features:

Raw view
Rendered view
bold, italic, underline, strikethrough, code block, inline code
headings (H1, H2, H3)
bullet lists, numbered lists, checklists
blockquotes
Mermaid charts

The app will start as a webapp:

Vite
React
Shadcn
TailwindCSS
The app will later be ported to native app with Tauri.

The default page should include a file drop, load or new file (prompt for new file name)
Open files will appear in tabs [Filename truncated 16...[x]]

Both the raw and rendered views will be editable.
The editor will have a split view with raw markdown on the left and rendered view on the right.
The editor will have a toolbar with buttons for the formatting options listed above in the preview.
The editor will support save as and export to pdf
Both of the views will have a menu bar with options to format text as listed above.

The app should support dark mode and light mode
The app should autosave changes every 30 seconds
The app should support lite mode and dark mode

The app should be dark, beautiful, and minimalistic.
The app should be responsive and work well on both desktop and mobile devices.
